Debug Interface

Keeps track of breakpoints set on the target, as well as deciding whether to use a hardware breakpoint or a software breakpoint.

Usage

const dbg = core.debug;

await dbg.setBreakpoint(0x123456);

// resume the core and wait for the breakpoint
await core.resume();
await core.waitForHalt();

// step forward one instruction
await dbg.step();

// remove the breakpoint
await dbg.deleteBreakpoint(0x123456);
